GetuiSDK(GexinSDK)
========

[个推 SDK](http://docs.getui.com/pages/viewpage.action?pageId=589866)

### Install

使用 [Cocoapods-depend](https://github.com/candyan/cocoapods-depend) 插件

``` pod depend add GetuiSDK ```

或者在 `Podfile` 文件下添加

``` pod 'GetuiSDK' ```

由于不太经常关注更新，如果发现有新版更新欢迎PR。Thx
